# Página de Soda Stereo

Es una página dedicada a la banda Soda Stereo, con un poco de información, y acceso a sus albumes y a las redes de sus integrantes.

## Herramientas usadas

En este proyecto se ha utilizado:

- Astro
- Tailwind
- Vercel
- Javascript
- HTML
- Css

## Cómo ejecutarlo

Entrar al deploy en [vercel](https://tp1-front-soda.vercel.app/). Si prefiere ejecutarlo de manera local tiene que seguir los siguientes pasos:

- Clonar el repositorio
- Dirigirse a la carpeta del repositorio y ejecutar ```npm install```
- Luego ejecutar ```npm run dev```
- Entrar a http://localhost:4321/.
